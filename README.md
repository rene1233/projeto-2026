# Sistema de Serviços Comunitários

## 📌 Descrição
Sistema web para solicitação de serviços na comunidade.

## 👥 Integrantes
- Seu nome + RA
- Nome dos colegas

## 🎯 Objetivo
Facilitar a solicitação de serviços para moradores e pequenos comércios.

## ⚙️ Funcionalidades
- Cadastro de usuário
- Login
- Solicitação de serviços
- Acompanhamento de status

## 🚀 Como usar
1. Fazer cadastro
2. Fazer login
3. Solicitar serviço
